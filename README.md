# PCF_net
**_PCF_net_** is an implementation of the paper _基于PCF-Net网络的建筑点云立面结构高精度提取_

> [!NOTE]
> We only provide the test code, the training code will be released later

## Requirement
pytorch > 1.9; Python > 3.8; cuda > 10.1
